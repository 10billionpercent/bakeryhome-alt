# Wagtail Bakery Homepage Alternative Look

A small alternate homepage redesign for the Wagtail Bakery demo site, built as part of my GSoC preparation.

## About
This project is a frontend exploration of the Bakery homepage with a different visual style, custom section components, and a more interactive hero area.

## What I Worked On
- rebuilt the homepage using modular JavaScript components
- experimented with custom styling and layout changes
- added a looping hero image conveyor for a more dynamic landing section

## Tech
- HTML
- CSS
- JavaScript
- Bootstrap

## Why I Made This
I used this project to practice working with an existing design, making thoughtful UI changes, and getting more comfortable with the kind of contribution and iteration work involved in GSoC prep.
